# 海克斯大乱斗助手

一个基于 AI 的英雄联盟海克斯大乱斗（ARAM）辅助工具，帮助玩家选择最佳的海克斯强化。

## 功能特点

- 🏹 **英雄选择**：支持 30 个常用 ARAM 英雄
- 💎 **海克斯选择**：包含银色、金色、棱彩三种品质的海克斯强化
- 🤖 **AI 推荐**：基于 Coze API 智能推荐最佳海克斯搭配
- 📱 **响应式设计**：支持桌面端和移动端

## 使用方法

1. 打开 `index.html` 文件
2. 点击右上角的"设置"按钮，输入你的 Coze API Key 和 Bot ID
3. 在英雄区域选择你当前使用的英雄
4. 在海克斯区域勾选你拥有的海克斯强化
5. 点击"获取推荐"按钮，等待 AI 分析结果

## 技术栈

- HTML5
- CSS3（响应式设计、渐变、动画）
- JavaScript（ES6+）
- Coze API（AI 推荐）

## API 配置

本项目使用 Coze（扣子）平台的 API：

1. 登录 [Coze 平台](https://www.coze.cn/)
2. 创建或选择你的 Bot
3. 获取 Bot ID 和 API Key
4. 在网页设置中输入配置信息

## 数据说明

### 英雄数据
- 包含 30 个常用 ARAM 英雄
- 每个英雄有名称、英文名、标签（射手/法师/坦克等）

### 海克斯数据
- 银色海克斯（8个）：基础增益效果
- 金色海克斯（7个）：中等强度增益
- 棱彩海克斯（5个）：强力增益效果
